About me

I am a fullstack developer specializing in front end UX/UI, by using an array of technologies such as: HTML, CSS, Javascript, React, MongoDB, Express, NodeJS. Coding started off as a hobby to an obsession; from making a html only websites, to making single page applications with React.I thrive in a team environment and value the opportunity to brainstorm and collaborate with my colleagues to yield client satisfaction. Web design has created a space for my creativity to flow—and with an open mind, sky’s the limit. Outside of coding I enjoy hiking, eating, and exploring the Northwest.
